**DeliveryOS** — SaaS de delivery do zero ao produto lançado

```
Painel web (restaurante)  →  Next.js + Tailwind + shadcn/ui
API central               →  NestJS + PostgreSQL + Redis
App do entregador         →  Expo + GPS + Socket.io (tempo real)
App do cliente            →  Expo + Mapas + Pagamentos
Serviço de IA             →  Python + FastAPI + LLMs
```
